# MERN Todo App

A full-stack Todo application built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This app allows users to register, log in, and manage their personal tasks with full CRUD functionality.

---

## Features

- User Registration
- User Login Authentication
- Secure Logout
- Create Todo
- Update Todo
- Delete Todo
- REST API Integration
- Responsive UI

---

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT / Cookies
